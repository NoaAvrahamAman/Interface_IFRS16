# Interface_IFRS16
Finance interface from Silvernet to Chemipal. One interface for 2 types of files: IFRS16, IFRS16_A

#######
CM_FIN_IN_IFRS16.xml - ION dataFlow. Reads file from folder and sends to MEC
IFRS16_File.xml - File Template
IFRS16_File.zip = Data object
IFRS16_AddCONO.json - Script to add CONO from path to content (also converts from windows-1255 to utf-8)
IFRS16_Transformed.zip - Data object for file after changes
CM_FIN_IN_IFRS16.zip - Files for Eclipse mapper. Extract zip before use in Eclipse
#######
